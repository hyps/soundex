How to use:
<br>
1. Install dependences (mongo, bottle, gevent, libevent, etc...)
<br>
2. Provide some data in config.py (db path, file path, etc...)
<br>
3. Run python web.py for web interface
<br>
4. Run python admin.py for basic command line interface
